# About
Some custom vim plugins and configurations.

# Set up
1. install git;
2. set up Vundle:<br/>
   `git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`
3. configure plugins:<br/>
   copy .vimrc to ~/.vimrc
4. install plugins:<br/>
   Launch vim and run :PluginInstall
